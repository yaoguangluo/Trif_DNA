Those metabolic optimizations were based on the software and its programmable files.

Definition of software secondary metabolism determinated a theory of evolutionary and PDN extension (PDE Encoding) optimizations of each indexed file name and Its function names, for instance of the meta Initon's PDE swap (AOPM -> VECS-> IDUQ), PDS (IDUQ -> VECS -> AOPM) swap and PDC encrypt (humanoid languages -> Initons encoding languages). And those instances could combinate an action of the first software metabolism at the same time.
